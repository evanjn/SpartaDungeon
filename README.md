# SpartaDungeon
Project Name : SpartaClub_Personal3DProject

스파르타클럽에서 진행하는 개인과제 프로젝트로, 유니티를 이용한 3D 게임을 강의를 바탕으로 실습한 게임입니다.

---
## ⌛ 개발기간
+ 내배캠 7~8주차, 11월 6일부터 11월 14일까지, 일주일 동안 작업한 프로젝트입니다.


---
## 🕹️ 게임 설명

닭을 조종해 아이템을 획득하며 허기를 채우거나 장착을 해서 치장을 해보세요!.
점프하며 높이 오르는 것이 목표입니다!.

플레이해서 도전해보세요!

---
## 🎈 조작법
+ WASD 키와 마우스 이동으로 방향을 조종할 수 있습니다.
+ space bar로 점프를 할 수 있습니다.
+ 닭 머리를 중심으로 마우스 휠을 이용해 회전하며 아이템과 닿으면 상호작용 자막이 나타납니다.
+ E키를 눌러 아이템을 획득 할 수 있습니다.
+ Tab키를 눌러 인벤토리 창을 띄워 각 아이템들을 사용하거나 장착할 수 있습니다.
+ 점프를 해서 점프대 위로 올라가 더 높이 점프를 할 수 있습니다.
